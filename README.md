# POKECAM — Web Version

## Setup (5 minutes)

1. **Copy your ROM** — rename your compiled file to `pokeblue.gbc` and put it in this folder alongside `index.html`

2. **Create a GitHub repo**
   - Go to github.com → New Repository
   - Name it `pokecam`
   - Set to **Public**

3. **Upload these files**
   - Drag `index.html`, `pokeblue.gbc`, and `README.md` into the repo

4. **Turn on GitHub Pages**
   - Go to repo Settings → Pages
   - Source: `Deploy from a branch`
   - Branch: `main` → `/ (root)`
   - Click Save

5. **Your game is live at:**
   `https://YOUR-USERNAME.github.io/pokecam`

## Link from Squarespace
Add a button on your Squarespace site pointing to that URL!
